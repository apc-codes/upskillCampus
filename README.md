# Upskill Campus Internship Training Projects

## Overview

This repository contains two Data Science projects developed using Python, Pandas, Matplotlib, Seaborn, and Scikit-learn.

---

# Project 1: Smart City Traffic Forecasting

## Objectives

- Analyze traffic data collected from four city junctions.
- Study traffic patterns during holidays and peak hours.
- Perform data preprocessing.
- Visualize traffic trends.
- Forecast traffic volume using Machine Learning.

### Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

# Project 2: Agriculture Crop Prediction

## Objectives

- Analyze crop production datasets across Indian states.
- Study production based on season, cultivation cost, and area.
- Clean and preprocess agricultural datasets.
- Perform exploratory data analysis.
- Predict crop production using Linear Regression.

### Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
